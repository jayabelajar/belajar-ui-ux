## 9.2 Tools untuk Riset & Analisis
| Tools | Fungsi | Keterangan |
|--------|---------|------------|
| **Google Forms / Typeform** | Survei pengguna | Mudah digunakan untuk riset awal |

## 9.8 Tips Memilih Tools
1. Pilih tools yang sesuai kapasitas tim (tidak harus mahal).  
2. Gunakan tools kolaboratif jika bekerja remote.  
3. Fokus pada *workflow*, bukan jumlah aplikasi.  
4. Integrasikan tools agar efisien antar tahap.  
5. Coba versi gratis sebelum berlangganan.
